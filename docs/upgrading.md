## Upgrading the site
When upgrading wagtail, check the:

- front page
  - news showing
- content search
- people finder search
- people finder
  - view profile
  - edit profile
  - view team
  - edit team
- django admin
- wagtail admin
  - home news order
    - drag to sort
  - add a news page
    - upload and add an image
    - publish and view live
    - appears on home page
    - appears on news page
